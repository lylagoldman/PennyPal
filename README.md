PennyPal is a student-focused expense tracking app built with Swift and SwiftUI. The goal of PennyPal is to help college students easily track daily spending and build healthier money habits in a simple, friendly interface.

This project is currently in progress. So far, the login screen UI has been implemented as the first step toward a full budgeting experience.

PennyPal was created as a learning project to practice iOS development while building something useful for college students managing tight budgets.
